## Quick Start | البداية السريعة

```json
{
  "mcpServers": {
    "arabic-accents": {
      "command": "npx",
      "args": ["-y", "arabic-accents-mcp"]
    }
  }
}
```

Add to `~/.cursor/mcp.json` (Cursor) or Claude Desktop config.

أضف إلى ملف `~/.cursor/mcp.json` (Cursor) أو إعدادات Claude Desktop.
