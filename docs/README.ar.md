# خادم MCP للهجات العربية

[![English](https://img.shields.io/badge/English-blue?style=flat-square)](README.en.md)

</div>

<div dir="rtl">

خادم [MCP](https://modelcontextprotocol.io) (بروتوكول سياق النموذج) مفتوح المصدر يوفر مكتبة من اللهجات والمفردات العربية. استخدمه مع أي وكيل ذكاء اصطناعي يدعم MCP (مثل Cursor أو Claude Desktop وغيرها) للحصول على سياق لترجمة أو تكييف المحتوى إلى اللهجة الإماراتية أو الشامية أو الخليجية أو غيرها من اللهجات العربية.

## ماذا يفعل

- **الموارد**: يوفر أدلة اللهجات (المفردات، العبارات، ملاحظات الاستخدام) كموارد MCP. يمكن لوكيل الذكاء الاصطناعي سحب سياق اللهجة المناسب عندما تطلب ترجمات (مثل "ترجم هذا إلى اللهجة الإماراتية").
- **قاعدة معرفية جاهزة**: يأتي مع بيانات اللهجات العربية جاهزة للاستخدام فوراً.
- **أي عميل**: يعمل مع Cursor وClaude Desktop وMCP Inspector أو أي تطبيق يدعم MCP.

## البداية السريعة (بدون تثبيت)

فقط أضف هذا إلى إعدادات عميل MCP الخاص بك — بدون استنساخ، بدون إعداد:

### Cursor

عدّل ملف `~/.cursor/mcp.json` (macOS/Linux) أو `%USERPROFILE%\.cursor\mcp.json` (Windows):

```json
{
  "mcpServers": {
    "arabic-accents": {
      "command": "npx",
      "args": ["-y", "arabic-accents-mcp"]
    }
  }
}
```

أعد تشغيل Cursor. انتهى.

### Claude Desktop

عدّل ملف `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) أو `%APPDATA%\Claude\claude_desktop_config.json` (Windows):

```json
{
  "mcpServers": {
    "arabic-accents": {
      "command": "npx",
      "args": ["-y", "arabic-accents-mcp"]
    }
  }
}
```

أعد تشغيل Claude Desktop. انتهى.

## الاستخدام

بعد الاتصال، اسأل مساعد الذكاء الاصطناعي:

- "ما هي اللهجات العربية المتاحة؟"
- "أرني مفردات اللهجة الإماراتية"
- "ترجم 'Hello, how are you?' إلى اللهجة الإماراتية"
- "حوّل هذا النص العربي الفصيح إلى لهجة خليجية عامية"

سيستخدم الذكاء الاصطناعي مكتبة اللهجات للتعبيرات الأصيلة.

### الموارد المتاحة

| المورد | الوصف |
|--------|-------|
| `arabic-accents://list` | قائمة بجميع اللهجات المتاحة |
| `arabic-accents://emirati` | مفردات وعبارات اللهجة الإماراتية |
| `arabic-accents://arabic-language` | أمثلة وبنية اللغة العربية |

### الأدوات المتاحة

| الأداة | الوصف |
|--------|-------|
| `get_accent_content` | الحصول على المحتوى الكامل للهجة معينة |
| `list_available_accents` | قائمة بجميع اللهجات المتاحة |

## المتطلبات

- Node.js 18+ (فقط للتشغيل المحلي)
- لاستخدام `npx`: يجب تثبيت Node.js، لكن لا حاجة لأي إعداد آخر

## التطوير المحلي

إذا أردت المساهمة أو تعديل الخادم:

```bash
# استنسخ المستودع
git clone https://github.com/Sumaida-ab/Arabic-context-MCP.git
cd Arabic-context-MCP

# ثبّت الاعتماديات
npm install

# ابنِ المشروع
npm run build

# شغّل الخادم
npm start

# استوعب ملفات PDF جديدة (عند إضافة لهجات)
npm run ingest
```
